DBFADD, DBFDEL and DBFFIND  - Immediate Instruction

Purpose

 The DFBADD, DBFDEL and DBFFIND instructions add, delete and retrieve information from the Jol Data Base of Data Sets.  This data base is accessed automatically when new data sets are created, and such information as record-format, space and volume information from the data base is automatically merged with new data set information.

Note:    These instructions are typically accessed through the Jol Menu system, and not directly by Jol Users.

However, they are also useful for converting old Jol programs to the new Jol format.



 

DBFADD       parameters normally specified on Data Set Declare

DBFDEL       'dsname'

DBFFIND      'dsname'

Menu Form: (Using Full Screen Data Entry Panels)

ADDDSN

DELDSN

Note:    When using the Menu Forms, you will be prompted for the required information.

      


Adding New Data Sets

Before Jol can merge data set information from the data base of data sets, the data set details must be placed in the data base.  The DBFADD instruction may be used to do this conveniently.

Deleting Data Sets

 Data sets can be deleted from the data base with the DBFDEL instruction.

Finding Data Set Information

 Often, users and Jol macros need to find information about data sets.  The DBFFIND instruction accesses the data base, and returns information in symbolic variables.

When the data set in found, the information is returned in the following symbolic variables:

Notes
  1. The interactive ADDDSN command is the preferred method of adding or altering information in the data base.
  2. You can use the SHOWDBF and SHOWDSN instructions to interactively select information from the Jol data base of data sets or the system catalog, respectively.
  3. All data set information is held in a VSAM data set on Mainframes.  Unix, Personal Computers hold the Data Base information in Data Bases appropriate for those machines.

Examples
1        DBFADD DSID1 DS MY.DATA.SET FB 80, 800
          VOL DISK02 10TRKS;

The data set name MY.DATA.SET is added to the Jol Data Set data base.

When the data set is created, Jol will automatically merge the information from the data base.  Thus, DCB, Volume and other information need not be specified in the Jol program, and all information about the data set is stored in only one place - the Jol data base.
