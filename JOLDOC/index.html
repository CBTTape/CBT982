This file (index.htnl) contains links to the current Jol manuals that have been converted to Html, RTFs or PDFs.  More will be converted from Spellbinder over time, and this page updated.

Click on the links with an X to open the manual in a new window, or tab if you have set your Web Browser to do that.

Note that some of the Html manuals automatically open parts of the manual in a new Frame, Tab or Window to make it convenient for you.  For example, if you select the page with a list of Instructions on it, then click on one of the instructions to show it's syntax and other information, then that detailed instruction will open in a new frame or page for your convenience.
