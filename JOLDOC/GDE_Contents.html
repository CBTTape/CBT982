Jol Reference Guide



This guide gives the User of Jol detailed descriptions 

of the Jol Universal Command Language
that enhances (or can be used in place of)
the IBM JCL (Job Control Language)
for Z/OS, VSE and other computing systems,
including Windows and Linux.

You can read more about Jol in the Jol Concepts and Facilities Manual.

Additionally, a quick index to the Jol Instructions may be found here.
