JOBPARM - Execute Command (Jes2 Only)

Purpose

 The JOBPARM Command specifies extra job information.  You may specify the expected elapsed time of your job, and other details like the maximum number of printed lines the job is allowed to produce.

 

JOBPARM     options ;

where options are:
   
CARDS
 = maxcards
COPIES = copies
ELAP or TIME
 = expected-elapsed-time
FORMS = form-number
LINECNT = lines-per-page
LINES
 = max-lines
ROOM = room-number
BOX
 = box-number
SYSTEM = * | ANY | system-number
NOLOG



where:


CARDS is the maximum number of cards the job is allowed to produce.
COPIES specifies the number of copies or any printed output that is to be produced.
ELAP or TIME specifies the expected elapsed time of the job, in minutes.
FORMS specifies print and punch forms on to which printed and punched cards are to be placed. If FORMS are specified on the PRINTER definition,it overrides the FORM specified here.
LINECNT specifies the maximum number of lines to be printed on a page. Code from 0 to 255.
LINES specifies, in thousands, the maximum number of printed lines the job will produce.
NOLOG specifies that you do not want a copy of the Jol and OS messages. Remember, that all Jol Catalog, Scratch,etc., messages will be lost if you specify NOLOG.
ROOM or BOX specifies data to be printed on the job's separator pages. This assists the operator to get results back to the programmer quickly.
SYSTEM specifies the systems to be eligible to process the job

* specifies the system which read the job.

ANY means any system in the JES2 multi-access spool

configuration.system number is a four digit number, specifying a particular system.



Notes
  1. Many of the details above may also be specified on the JOB statement. Any details specified in the JOBPARM command override those used on the Job statement.
  2. If the JOBPARM command is executed on a JES3 or ASP system, the MAIN command will be executed with the parameters specified in the JOBPARM command.
  3. The JOBPARM command may be coded anywhere in the Jol program.

Examples

JOBPARM ELAP 10, COPIES 2,

ROOM 1024;

The JOBPARM command specifies that the job is expected to run for 10 minutes. Two copies of any output are requested to be sent to Room 1024.
